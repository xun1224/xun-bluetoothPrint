# xun-bluetoothPrint

可以用于小程序和uniapp app的蓝牙打印通用版本

```
<template>
    <view>
        <xun-bluetoothPrint></xun-bluetoothPrint>
    </view>
</template>

<script>
import xun_bluetoothPrint from '../../components/xun-bluetoothPrint/index.vue'
export default {
    components: {
        'xun-bluetoothPrint': xun_bluetoothPrint
    }
}
</script>
```
